# Security policy

## Supported versions

Jasper Media Player is an early project. Security fixes are handled on the latest release only.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a vulnerability

Please open a private report if GitHub security advisories are available for this repository. Otherwise, open an issue with a clear description and avoid posting exploit details that could harm users.

Include:

- App version or commit
- Windows version
- Steps to reproduce
- What happened
- What you expected to happen

## Media URL warning

Only open media URLs you trust. Network streams are handled by media playback components and can depend on third-party servers.
